# 排序算法
## <span id="冒泡">冒泡排序（Bubble Sort）</span>

>**定义**：重复地走访过要排序的元素列，依次比较两个相邻的元素，如果他们的顺序（如从大到小、首字母从A到Z）错误就把他们交换过来。走访元素的工作是重复地进行直到没有相邻元素需要交换，也就是说该元素列已经排序完成。

![Bubble](/assets/images/post/2019-08-05/bubble.gif)


    #include<iostream>
    using namespace std;

    void BubbleSort(int arr[], int len);

    int main(){
        int arrays[] = {1, 5, 3, 9, 6, 4, 7, 2, 8};
        int length = sizeof(arrays)/sizeof(arrays[0]);
        BubbleSort(arrays, length);
    }

    void BubbleSort(int arr[], int len){
        bool swapped = false;
        do{
            swapped = false;
            for (int i = 0; i < len; i++){
                if (arr[i] > arr[i+1]){
                    swap(arr[i], arr[i+1]);
                    swapped = true;
                }
            }
        }while(swapped);
    }
